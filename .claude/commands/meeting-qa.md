Прочитай запись звонка из текстового файла по пути: $ARGUMENTS

Сделай из неё структурированный документ в формате «Вопрос — Ответ».

## Правила

1. **Атрибуция**: каждый вопрос и каждый ответ должны быть подписаны именем участника, который их произнёс.
2. **Точность**: не додумывай. Используй цитаты или их фрагменты из расшифровки где возможно, но делай текст читаемым — убирай слова-паразиты, оговорки и повторы, лишнее.
3. **Полнота**: не теряй важных деталей — решения, договорённости, открытые вопросы, цифры, сроки.
4. **Компактность**: убирай smalltalk, приветствия, прощания и нерелевантные отступления, логические повторы.
5. **Формат вывода**: markdown-документ. Каждая пара «вопрос-ответ» — отдельный блок. Если на один вопрос отвечают несколько человек, приведи все значимые ответы.

## Формат блока

```
### <Краткая тема вопроса>

**Вопрос (<Имя>):** <текст вопроса>

**Ответ (<Имя>):** <текст ответа>
```

Если в обсуждении нет явной структуры «вопрос-ответ», но есть обсуждение темы — сгруппируй по темам и сохрани ключевые высказывания участников в том же формате, пометив вопрос как тему обсуждения.
